# Property Search App (CakePHP 5)

A simple property listing and search application built with CakePHP 5.

## Features
- Add properties with multiple photo uploads
- Automatic image resizing
- Property search
- Property detail pages with photo gallery
- Responsive grid layout
- Lightbox photo viewing

## Tech Stack
- PHP 8.2
- CakePHP 5
- MariaDB
- Intervention Image (GD)
- GLightbox

## Setup Instructions
1. Clone the repo
2. Run `composer install`
3. Configure database in `config/app_local.php`
4. Run migrations / import SQL ('property_search.sql')
5. Visit url of setup
